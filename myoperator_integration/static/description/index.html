MyOperator Integration

Connect Odoo with MyOperator for seamless call and WhatsApp integration

Key Features

  • Connect Odoo with MyOperator's API
  • Sync call logs from MyOperator to Odoo
  • Sync WhatsApp messages from MyOperator to Odoo
  • Make outbound calls directly from Odoo
  • Send WhatsApp messages directly from Odoo
  • Link customers with their communication history
  • Automated synchronization with scheduled jobs
  • Real-time updates via webhooks

Easy Configuration

Set up the integration with just a few steps:

  1. Obtain API token from MyOperator panel
  2. Configure the API settings in Odoo
  3. Test the connection
  4. Start synchronizing your communication history

Call History Integration

  • View all call logs in one place
  • Filter by call type, status, and date
  • Link calls to contacts automatically
  • Listen to call recordings
  • Add notes to calls for future reference

WhatsApp Integration

  • Send WhatsApp messages to contacts
  • Track message delivery status
  • View complete message history
  • Support for text and media messages
  • Bulk messaging capability
